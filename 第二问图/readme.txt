1：三个中文名图片需要做成表格，其中测试值和预测值放在一起做对比，测试值较少对应预测值前几个。
2：异常值分析可以从标准差std和均值mean差距不大，最大值、最小值之间在可接受范围，4分位值也正常来做说明，
    可以每个变量都做一些描述凑字数
3：aqi-predict是使用自己找的的数据集通过搭建神经网络完成，该图片为测试集上的预测值和实际值关系散点图，可以
    观察到具有一定效果，以此模型对附件1每小时实测数据进行预测，为该数据添加一个aqi的属性列
4: 5张环境图都是和aqi的关系散点图，但这不是直接关系，而是采用了3天为间隔，将原始数据每三天的数据做差
    （如：2020/1/3/24:00  - 2020/1/1/24:00）。即三天的72条数据最终为1条数据，以此得到环境的变化情况，（之所以选择3天是考虑到环境情况具有一定的周期性，需要去除其周期性的影响，选择3天则是保证做差之后数据具有足够的差异）
最后可以观察到环境的变化以及aqi的变化之间的关系。如图temp，当x轴temp增加，aqi呈现的也是增长的趋势。可以更具其趋势的剧烈程度对环境进行分类，例如温度变化明显就每增加或减小5作为1类，风向变化弱就大一些。这里自由发挥。
另外更具两个柱状图可以发下，哪怕是三天的间隔也有大量的数据并没有展现出明显的差异，变化量仍然是集中在0左右，而偏离直线的点则是因为aqi值还受到其他因素的干扰（稍作说明即可）

对于风向图，不能说其对aqi没什么影响，只是相对于其他的因素来说，因为每张图片的因素自变量环境单一，但aqi值却是受到多变量的影响，可以说其与其他因素相比，对aqi的影响最小。

注：由题目，这一切都是在排放浓度不变的情况下完成。但实际上数据却不是这样，实际数据的aqi受到的最大影响因素还是来自污染物排放的浓度。
